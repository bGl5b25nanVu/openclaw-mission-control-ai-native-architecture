# 09 Human-in-the-Loop Boundaries

## Principle
Humans should not be the default execution engine.
Humans should intervene mainly for:
- strategic reprioritization
- approval of risky/external actions
- policy conflicts
- repeated failed loops
- ambiguous trade-offs between goals

## AI default scope
AI should normally:
- observe signals
- create or continue runs
- gather evidence
- update loop state
- suggest escalations

## Escalation triggers
- high external impact
- payment / legal / public communication risk
- repeated failed runs without new evidence
- insufficient permissions or missing external input
- conflicting policies between loops
